# Question Part
This parameter is stored in the [`parts`][1] list in a [question][2] JSON file. It is the primary unit of content inside a question, holding:

 - [part statement][3]
 - [response area][4]
 - [media][5]
 - [final answer][6]
 - [worked solutions][7]
 - [structured tutorial][8]
